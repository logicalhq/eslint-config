## Getting Started

Make sure you have [eslint](https://github.com/eslint/eslint) and [eslint-plugin-import](https://github.com/benmosher/eslint-plugin-import) installed, then install the plugin:

```bash
yarn add -D @logicalhq/eslint-config-base
```

Don't forget to also update your `.eslintrc` configuration file:

```
{
  "extends": "@logicalhq/eslint-config-base"
}
```
